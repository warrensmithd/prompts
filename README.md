# 1. Systems Decomposition Framework

Best for:

* technical essays
* AI architecture
* infrastructure
* organizational systems
* economics

This extracts:

* entities
* flows
* dependencies
* bottlenecks
* incentives

### Prompt Pattern

```text
Analyze the article as a system.

Extract:
• Core actors/entities
• Inputs
• Outputs
• Incentive structures
• Bottlenecks
• Feedback loops
• Failure modes
• Scaling constraints
• Hidden assumptions

Return results in a table.

Columns:
System Element | Description | Dependency | Risk | Scaling Impact

Rules:
• Focus on causality
• Avoid summaries
• Prefer mechanisms over opinions
• Identify invisible dependencies
• Abstract transferable patterns
```

Why it works:
This is extremely effective for:

* distributed systems
* AI infrastructure
* organizational theory
* sociology
* economics
* platform/business models

It forces the model to think mechanistically.

# 2. Compression → Expansion Prompt

Best for:

* dense essays
* philosophy
* sociology
* intellectual history
* long-form think pieces

### Prompt Pattern

```text
Analyze the article through progressive compression.

Return:
1. One sentence thesis
2. Three core ideas
3. Five supporting mechanisms
4. Ten transferable principles
5. Three predictions implied by the article
6. Three hidden assumptions
7. Three strongest counterarguments

Rules:
• Preserve nuance
• Avoid generic summaries
• Extract reusable reasoning patterns
• Focus on causal structure
```

Why it works:
This creates:

* executive summaries
* podcast prep
* speaking prep
* strategic memory anchors

Excellent for Substack-style essays.

# 3. Operator Extraction Framework

Best for:

* startup essays
* PM/product content
* leadership writing
* execution articles
* AI workflow essays

### Prompt Pattern

```text
Extract only operationally actionable insights from the article.

For each insight provide:
• Situation
• Signal
• Decision
• Tradeoff
• Execution tactic
• Failure risk
• Organizational impact

Rules:
• No motivational language
• No abstract theory
• Focus on repeatable execution behaviors
• Prefer specific operating patterns
```

This is one of the highest-value frameworks for product leaders.

It converts intellectual content into:

* management heuristics
* execution playbooks
* organizational patterns

# 4. Ideological Mapping Framework

Best for:

* sociology
* political essays
* economics
* cultural criticism
* AI/social impact essays

### Prompt Pattern

```text
Analyze the article ideologically.

Extract:
• Explicit worldview
• Implicit assumptions
• Human nature assumptions
• Economic assumptions
• Institutional assumptions
• Technological assumptions
• Power structures referenced
• Historical parallels
• Competing schools of thought

Then identify:
• What the author likely overestimates
• What the author likely underestimates
```

Why it works:
Most sociology/political writing is really:

* assumptions about incentives
* assumptions about power
* assumptions about institutions

This framework surfaces them directly.

# 6. First-Principles Reconstruction

Best for:

* philosophy
* sociology
* economics
* speculative theory
* scientific essays

### Prompt Pattern

```text
Reconstruct the article from first principles.

Identify:
• Foundational assumptions
• Primitive concepts
• Derived conclusions
• Logical chain of reasoning
• Necessary conditions
• Sufficient conditions
• Contradictions
• Unproven leaps
• Areas requiring empirical validation

Then rebuild the argument in:
1. Simplest form
2. Strongest form
3. Most falsifiable form
```

This is extremely powerful for:

* intellectual rigor
* theory testing
* identifying weak logic

# 7. Narrative → Incentive Translation

Best for:

* sociology
* economics
* management
* media analysis
* cultural essays

### Prompt Pattern

```text
Translate the article from narrative language into incentive language.

For each major point identify:
• Actor
• Incentive
• Constraint
• Risk
• Reward
• Coordination problem
• Information asymmetry
• Economic driver

Rules:
• Reduce emotional framing
• Focus on incentives and systems
• Identify hidden coordination structures
```

This is incredibly effective because most social systems are incentive systems disguised as narratives.

# 8. Cognitive Load Flashcard Framework

A stronger alternative to traditional flashcards for dense essays.

### Prompt Pattern

```text
Convert the article into high-retention cognitive compression cards.

Each card must contain:
• Core idea
• Why it matters
• Mechanism
• Memorable analogy
• Real-world application
• Counterexample

Limit:
<300 characters per card
```

This improves:

* memory retention
* speaking fluency
* teaching
* strategic recall
